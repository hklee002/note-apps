# note-apps
(markdown) note web application

## The Technology Stack
### Server
- express
- Node
- [pockeddb](https://github.com/samisking/PocketDB)

### Client
- React
- Flux Architecture with Redux

## Installation
1. install node.js
2. git clone https://github.com/hklee002/note-apps.git
3. cd note-apps
4. npm install
5. cd api
6. npm install 
7. cd..
8. npm run api
9. npm start
